HackerArchive – Kompletny tekst do nauki C++

Ten dokument jest jednym, długim tekstem przeznaczonym do nauki języka C++ – od absolutnych podstaw aż po bardziej zaawansowane koncepcje. Czytaj go liniowo, jak książkę. C++ jest językiem niskopoziomowym, szybkim i potężnym, używanym w systemach operacyjnych, silnikach gier, oprogramowaniu embedded i aplikacjach wysokiej wydajności.

1. Czym jest C++

C++ to język kompilowany, co oznacza, że kod źródłowy musi zostać przetworzony przez kompilator (np. GCC, Clang, MSVC) do kodu maszynowego. Programy w C++ są bardzo szybkie, ale wymagają od programisty odpowiedzialności za pamięć.

2. Pierwszy program

Klasyczny program Hello World wygląda następująco:

#include 

int main() {
    std::cout << "Hello, world!" << std::endl;
    return 0;
}

Funkcja main jest punktem wejścia programu. #include dołącza bibliotekę, a std::cout służy do wypisywania tekstu.

3. Zmienne i typy danych

C++ jest językiem statycznie typowanym. Oznacza to, że każda zmienna ma określony typ: 

int a = 10;
double b = 3.14;
char c = 'A';
bool isTrue = true;

Typy podstawowe to m.in. int, float, double, char i bool.

4. Instrukcje warunkowe

Warunki pozwalają podejmować decyzje w kodzie: 

if (a > 5) {
    std::cout << "a jest większe niż 5";
} else {
    std::cout << "a jest mniejsze lub równe 5";
}

5. Pętle

Pętle służą do wielokrotnego wykonywania kodu: 

for (int i = 0; i < 10; i++) {
    std::cout << i << std::endl;
}

while (a > 0) {
    a--;
}

6. Funkcje

Funkcje pozwalają dzielić program na mniejsze części: 

int add(int x, int y) {
    return x + y;
}

7. Wskaźniki i pamięć

Jedną z najważniejszych (i najtrudniejszych) części C++ są wskaźniki. Wskaźnik przechowuje adres w pamięci: 

int x = 5;
int* ptr = &x;

Nieprawidłowe użycie pamięci może prowadzić do błędów krytycznych, dlatego C++ wymaga ostrożności.

8. Programowanie obiektowe

C++ obsługuje klasy i obiekty: 

class Person {
public:
    std::string name;
    int age;

    void sayHello() {
        std::cout << "Cześć, mam na imię " << name;
    }
};

9. Standardowa biblioteka

STL (Standard Template Library) oferuje kontenery (vector, map), algorytmy i narzędzia, które znacząco ułatwiają pisanie kodu.

10. Zakończenie

C++ to język trudny, ale niezwykle potężny. Nauka wymaga czasu, praktyki i zrozumienia tego, co dzieje się „pod spodem”. Jeśli opanujesz C++, zrozumiesz też, jak działa większość innych języków.
