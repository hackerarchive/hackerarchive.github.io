1. Czym jest Python?

Python to nowoczesny, interpretowany język programowania, który słynie z prostoty i czytelności. Używany jest w cyberbezpieczeństwie, automatyzacji, analizie danych, AI, web developmencie i hackingu.

2. Pierwszy program

Każdy język zaczyna się od prostego przykładu. W Pythonie wygląda to tak: 

print("Hello, world!")

Funkcja print wypisuje tekst na ekranie.

3. Zmienne

Zmienne służą do przechowywania danych. Python nie wymaga deklarowania typu. 

x = 10
y = "tekst"
pi = 3.14

4. Typy danych

Najważniejsze typy danych w Pythonie:

  • int – liczby całkowite
  • float – liczby zmiennoprzecinkowe
  • str – tekst
  • bool – True / False

5. Warunki

Instrukcje warunkowe pozwalają podejmować decyzje. 

age = 18

if age >= 18:
    print("Dostęp przyznany")
else:
    print("Brak dostępu")

6. Pętle

Pętle pozwalają wykonywać kod wielokrotnie. 

for i in range(5):
    print(i)
x = 0
while x < 5:
    print(x)
    x += 1

7. Funkcje

Funkcje pozwalają organizować kod. 

def greet(name):
    return "Cześć " + name

print(greet("Anon"))

8. Listy

Listy przechowują wiele wartości. 

tools = ["nmap", "sqlmap", "hydra"]

for tool in tools:
    print(tool)

9. Słowniki

Słowniki przechowują dane w parach klucz-wartość. 

user = {
    "login": "admin",
    "password": "1234"
}

print(user["login"])

10. Praca z plikami

with open("data.txt", "w") as f:
    f.write("Hello")

11. Moduły

Python posiada tysiące modułów. 

import os
print(os.getcwd())

12. Python w praktyce

Python jest często używany do:

  • automatyzacji zadań
  • tworzenia skryptów security
  • analizy logów
  • web scrapingu
  • AI i ML
